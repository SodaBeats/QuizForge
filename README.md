QuizForge
-A teacher-first web application that accepts a pdf or docx document, choose between "generate" or "manual". The former would use an LLM to review the document
and generate questions based on question type (choices: multiple choice, true or false, short answer). 
The teacher can then share that quiz using a sharing token. The students can log in and input the quiz token, where they would be redirected
to the quiz page. Once the students are done, an automated score calculation happens in the server and records the score for that quiz attempt.
